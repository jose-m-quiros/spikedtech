# SPIKEDTECH - Sitio corporativo

SPIKEDTECH es un sitio estático orientado a presentar servicios de desarrollo web, sistemas a medida, integraciones y continuidad técnica para negocio.

## Estructura actual

- index.html: portada principal
- servicios.html: oferta general
- desarrollo-web.html: detalle del servicio de desarrollo
- proyectos.html: referencias y formatos de solución
- contacto.html: formulario comercial y canales de contacto

## Desarrollo local

No hay proceso de build ni dependencias de instalación.
Puedes abrir index.html directamente en el navegador o servir la carpeta con cualquier servidor estático.

## Estado del repositorio

El repositorio conserva únicamente las páginas activas del sitio, sus dos imágenes en uso, la hoja de estilos principal, el script del frontend y los archivos de configuración básicos.
